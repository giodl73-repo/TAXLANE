# TAXLANE Wave

Use this skill to create or update TAXLANE waves.

## Wave requirements

- Separate descriptive tax-system research from reform-design claims.
- Identify affected docs, research notes, datasets, or public reading packets.
- Break the wave into small pulses.
- Include validation expectations for every pulse.
- Record non-goals and source-custody requirements.
