# Program-Lane Rate Charts

Charts for the program-lane operating system.

| Chart | Purpose |
|---|---|
| `lane-rate-bar.vl.json` | Horizontal bar of each lane's cents-per-dollar of FY2025 outlays, colored by funding source. Data: `lane-shares.csv`. |

These are reform-model visualizations. Lane shares are modeled allocation, not
legal dedication. About 25c of every dollar is borrowed, not collected; see the
explainer `docs/reading/program-lane-system.md`.
