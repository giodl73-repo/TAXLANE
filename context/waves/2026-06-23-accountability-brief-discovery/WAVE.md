# Wave: Accountability Brief Discovery

## Objective

Make the accountability public brief discoverable from reader entry points.

## Context

The public brief is useful only if readers can find it. Link it from the
top-level README and reading packet index, and validate those links.

## Pulses

| Pulse | Name | Status | Notes |
|---|---|---|---|
| 01 | Brief discovery | done | Linked the public brief from README and reading index, added discovery validation, review, manifest coverage, and VTRACE closeout. |

## Acceptance

- Top-level README links the public brief.
- `docs/reading/README.md` lists the public brief and includes it in reading order.
- Validation fails if discovery links are removed.
- Reading guardrails preserve question-surface boundaries.
