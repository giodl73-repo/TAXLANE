# Wave: Accountability Public Questions

## Objective

Generate a public-safe question packet for accountability evidence blockers.

## Context

The claim guard confirms no current accountability record supports a public
claim. TAXLANE still needs a safe outward-facing artifact: questions people can
ask to demand reviewed evidence, without repeating draft evidence as findings.

## Pulses

| Pulse | Name | Status | Notes |
|---|---|---|---|
| 01 | Public-safe questions | done | Added generated public questions, stale-output validation, review, manifest coverage, and VTRACE closeout. |

## Acceptance

- Packet lists only questions and blockers.
- Packet excludes full evidence rows and public claim assertions.
- Packet states that it is not a finding surface.
- Validation fails if the packet is stale.
